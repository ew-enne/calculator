	- entryArray = ['0']
	- operatorArray = []
	- operator

	-> reset everything


(1) enter numbers :

	-> populate entryArray

	-> set firstEntry = true

	-> empty entryArray

(3) enter numbers :

	-> define operator
	-> populate entryArray
	-> empty operatorArray

	-> populate operatorArray
	-> define secondNumber

	-> show result on screen
	-> define operator
